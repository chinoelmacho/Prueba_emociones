# Prueba de emociones

Pequeño paquete de referencia para entrenar y evaluar un modelo clásico de
clasificación de intensidad emocional. Usa un pipeline TF-IDF + regresión
logística y está pensado para experimentar rápidamente con datasets propios
o de Hugging Face.
